ChatLogic Vector Database

The overarching objective of this initiative is to revolutionize the technical support operations at Shocklogic Company by developing and implementing a vectorial database that will significantly boost the conversational prowess of the existing chatbot. Currently, the chatbot operates on a SQL database and utilizes a relatively rudimentary search algorithm to answer queries. This search algorithm functions by matching labels stored in a database table with the keywords or phrases present in the user's input. While functional, this approach has its limitations in terms of adaptability and the ability to understand nuanced queries. By transitioning to a vectorial database, it is aimed to leverage more advanced computational techniques to interpret natural language, thus delivering more precise, contextually relevant responses to the users. This transformation will not only improve the customer experience but also streamline the process for the technical support team by reducing the likelihood of errors and misunderstandings.

Trade Tracker

It's a tool designed to streamline and optimize the trading experience within the Discord community. This bot offers a suite of features focused on enabling traders to simulate trades, check real-time prices, set alerts, and receive important market updates. Prioritizing user engagement and utility, this bot aims to be an essential companion for traders.

Features:
  • Paper Trading (futures and spot): Users can open a simulated or "paper" futures trading position through the bot. Notifications will be triggered when the position hits predefined take-profit (TP) and stop-loss (SL) levels.
  • Price Checker: Users can check the real-time price of any token by using a specific command within Discord.
  • Price Alerts: Allows users to set up alerts that trigger when a price rises above or falls below a predetermined level.
  • Trading Reports: Store users' trading history and display relevant statistics. Reports will include monthly breakdowns featuring each trade's pair, leverage, margin, profit and loss (PNL), and return on investment (ROI). Cumulative ROI and PNL will also be shown for each month.
